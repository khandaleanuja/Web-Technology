SQL Basics
Important Commands

1. CREATE TABLE
CREATE TABLE students (
  id INT PRIMARY KEY,
  name VARCHAR(50),
  age INT
);

2. INSERT DATA
INSERT INTO students VALUES (1, 'Anuja', 22);

3. SELECT (Fetch Data)
SELECT * FROM students;

4. UPDATE
UPDATE students SET age = 23 WHERE id = 1;

5. DELETE
DELETE FROM students WHERE id = 1;


Connecting Node.js to SQL Database :

We use Node.js backend to connect database.
Popular runtime:
•	Node.js
Package used:
npm install mysql2

Example Code
const mysql = require('mysql2');

const connection = mysql.createConnection({
  host: 'localhost',
  user: 'root',
  password: 'your_password',
  database: 'college'
});

connection.connect((err) => {
  if (err) throw err;
  console.log('Connected to MySQL');
});



Query Execution in Node.js
Example: Fetch Data
connection.query("SELECT * FROM students", (err, result) => {
  if (err) throw err;
  console.log(result);
});
Insert Data
connection.query(
  "INSERT INTO students (id, name, age) VALUES (2, 'Amit', 21)",
  (err, result) => {
    if (err) throw err;
    console.log("Data Inserted");
  }
);

ORM means:

Popular ORM:
