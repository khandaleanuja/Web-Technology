   (> < >= <=) convert values to Number
   null -> Number(null) -> 0
   0 > 0 --> false

-> Output : false
   null is only loosely equal to undefined. It does not convert to number here
   null == 0 --> false

console.log(null <= 0)
-> Output : true
   null -> Number(null) -> 0
   0 <= 0 --> true
